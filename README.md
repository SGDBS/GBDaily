## 如何操作
### git操作
第一次使用:
```bash
git clone git@github.com:SGDBS/GBDaily.git
```

之后进行修改:
```
git fetch origin master:temp
git merge temp
```
用于更新到最新版本

```bash
git add ./
git commit -m "写你干了啥"
git push origin master
```
在你添加完东西后,在仓库的根文件夹执行
